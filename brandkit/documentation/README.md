# Pulse — Brand & Design Package

Complete brand identity and design package for **Pulse**, the self-hosted real-time
analytics, QoE monitoring and alerting module for Ant Media Server.

## Folder structure

```
/brandkit          Brand Guidelines (open "Brand Guidelines.dc.html")
/logo              Logo suite — SVG masters + PNG favicons/marks
/icons             App icon concepts + iOS/Android/web variants + PNG exports
/design-system     Component library (light+dark) + tokens.json
/ui                Hi-fi product screens (login → mobile), canvas view
/website           Marketing website (single responsive page)
/assets            OG image, social square, deck cover, email signature, patterns
/documentation     Design rationale, accessibility, UI guidelines, roadmap
index.dc.html      Start here — links to everything
```

## Quick facts

- **Primary theme:** dark (`#0A0E14` base), light theme included
- **Signal color:** `#2CE5A7` — means live/healthy/primary-action only
- **Type:** IBM Plex Sans + IBM Plex Mono (OFL — self-host in production)
- **Logo:** single-stroke pulse line; favicon-legible at 16 px; white-label badge included
- **Tokens:** `design-system/tokens.json` (hex, spacing, radii, type scale)

All `.dc.html` files open directly in a browser. SVGs are the editable masters;
PNGs are exports. See `documentation/design-rationale.md` for decisions and
accessibility verification.
