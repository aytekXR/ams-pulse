2. **QoE dashboard (F3 surface, `/qoe`):** summary cards (startup p50/p95,

   table + choropleth-free ranked bars — no external map tiles, LAN rule);
   devices tab (device/os/browser/protocol breakdowns); both sliceable and
   CSV-exportable per spec.
5. **Reports (F6, `/reports`):** usage view (per app/stream/tenant table:
   viewer-minutes, peak concurrency, egress GB with method tooltip, recording
   GB; date range); statement generation + download (CSV/PDF); schedules CRUD
   (cron presets UI: monthly/weekly + raw cron field); tenants CRUD (pattern/
   tag mapping with live preview of matched streams); white-label header
   config. Gate-aware: Business-tier upsell state when entitlement missing
   (read license tier from API, render upgrade prompt, never a broken page).
6. **Fleet (F7, `/fleet`):** node cards/table (role badge origin/edge, status,
   last_seen, version, cpu/mem/net load bars); aggregate header with dedup
   note; node history sparkline if cheap. Auto-refresh.
7. **Settings additions:** ingest tokens management surface (create/revoke for
   beacon SDK, copy-paste snippet showing Pulse.init with the token);
   Prometheus endpoint info panel (URL + optional token state); S3 export
